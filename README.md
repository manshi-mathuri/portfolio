# Personal Portfolio Website

A modern and responsive **Portfolio Website** built using **Next.js, TypeScript, CSS, and JavaScript**.  
This project showcases my skills, projects, education, and technical experience with a clean and professional user interface.

## Features

- Responsive portfolio design
- About Me section
- Skills showcase
- Projects section
- Contact section
- Smooth navigation
- Modern UI/UX design
- Optimized performance with Next.js

##  Tech Stack

- Next.js
- TypeScript
- CSS
- JavaScript
- HTML

## Project Description

This Portfolio Website represents my personal and professional journey as a Computer Science Engineering student and web developer. The website highlights my technical skills, projects, achievements, and contact information in a structured and visually appealing format.
